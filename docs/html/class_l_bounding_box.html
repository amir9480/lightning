Detailed Description

Axisally Aligned Bounding Box.

This class is for Axisally Aligned Bounding Box and using for intersections and visability determinate

whereas AABB is computed by two point Min and Max, but if you need other corners you can use LBoundingBox::getCorner

Max corner is LCorner::right_top_front and min corner is LCorner::left_bottom_back
