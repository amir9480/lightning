Detailed Description

Vector 4D class.

This class using to define 4D vectors instead of 3D vectors and recommended for Multiply with matrix

Note
To convert to 3D vector , all parameter must division on W

LVector3(x,y,z) = LVector4(x/w,y/w,z/w,w/w) -> w=1

LVector4 a(1.0f,2.0f,3.0f);
a=a*LMatrix::createTranslationMatrix(LVector3(5.0f,0.0f,0.0f));
